Upload Custom Fragment
Create a fragment with images from your computer or the internet. First, select images for recto and/or verso. If you have a two-sided object, you use the mask & registering tools to register front and back (which allows fragment and table flips). For correct scaling and measurement, you need to give information on the image resolution. If not extracted automatically, you can either type the PPI into the respective fields or use the measure tool to define the distance of 1cm within the image.
Fields in red colour are obligatory before you can add your fragment to the table.
